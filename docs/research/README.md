# Research

Findings from experiments, benchmarks, and external-docs deep-dives.

## What goes here

- whisper.cpp benchmark results (HR accuracy, speed, model comparisons)
- Apple API investigations (Accessibility, global hotkey, AVAudioEngine)
- Comparison matrices that feed into a decision
- Short write-ups of failed approaches (so we don't repeat them)

## What does NOT go here

- Decisions themselves → `docs/decisions/`
- Phase plans → `docs/specs/`
- Code → `src/`

## File naming

Free-form but descriptive. `whisper-hr-benchmark-2026-04-21.md`, `accessibility-api-notes.md`.
